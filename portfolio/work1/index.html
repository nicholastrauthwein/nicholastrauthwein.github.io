California Housing Prices Analysis

This Excel dashboard is an analysis of Kaggle data on home prices in California from 1990. This dashboard includes a single page breakdown of the summary statistics and pivot charts, heatmaps, and other visualizations that summarize the important findings from the data. The second sheet contains the actual data table that was downloaded from Kaggle and manipulated in Excel.
