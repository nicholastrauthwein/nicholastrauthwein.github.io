Target Stock Dashboard

This project is a stock performance dashboard created using Microsoft Excel. The company of interest in this project was Target (TGT), and KPI's analyzed included return, market capitalization, and P/E ratio. The dataset used was obtained using Yahoo Finance and the dashboard is designed such that it can be continually updated and reflect these changes for further insights.
